Introduction/Background

Project 2 focused on developing GT Marketplace, a platform for Georgia Tech students to buy and sell items within their community. The project utilized a modern technology stack combining React for the frontend and Flask for the backend, with Google Cloud Platform services for deployment and storage.

The marketplace addresses the need for a dedicated, secure trading platform specifically for the GT community. Our project management approach followed an agile methodology with 2-week sprints and regular stand-ups to ensure consistent progress.

Unlike existing solutions like Facebook Marketplace or Craigslist, GT Marketplace provides verified student-only access through Google OAuth, ensuring a trusted environment for transactions within the campus community.

Software Technologies

Frontend

  • React.js with Vite for enhanced development experience
  • Bootstrap 5 for responsive design
  • React Router for client-side routing

Backend

  • Flask for RESTful API development
  • Google Cloud Storage for image handling
  • Firebase Admin for authentication
  • JWT for secure token management
